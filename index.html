Comment mettre en place un site web avec une raspberry pi ?

Mise en place d'un "LAMP"

Après avoir mis en place son raspberry pi, il faut installer un LAMP ( pour linux, apache, mysql et php). Suivre le tuto Installer un serveur web sur votre Raspberry (Apache + PHP + MySQL) Puis la suite de ce tuto : Rendre votre Raspberry Pi accessible depuis internet avec DynDNS et le port Forwarding. Sur une freebox, il est possible de mettre en place d’une adresse IP fixe (gratuit).

Mise en place d'un "DNS"

Une fois cela fait, il faut mettre en place un DNS, disponible gratuitement chez free.

Sécuriser son site web avec certbox !

Pour cela rien de plus simple, suivre le tuto se trouvant à l'adresse : https://certbot.eff.org/lets-encrypt/debianbuster-apache

Modifier son site web depuis n'importe où avec SSH et VSC

Mettre en place SSH

https://www.dreamhost.com/blog/learn-how-to-use-ssh/

Se conecter à distance

Depuis un terminal sous la machine distante, taper : $ ssh pi@dtresiea.hd.free.fr Entrer le mot de passe et la raspberry pi est accessible et modifiable. Pour utiliser un IDE sous la machine distante, installer VSC et l’extension ssh : https://code.visualstudio.com/docs/remote/ssh

Pouvoir modifier les fichiers sans être en sudo

Pour avoir le droit de modifier les fichiers, taper dans le terminal en ssh ou directement depuis la raspberry pi : $ sudo chown -R pi /var/www (https://github.com/microsoft/vscode-remote-release/issues/1008) Ainsi, tous les fichiers du site web seront modifiable depuis l’utilisateur pi sans être en root depuis VSC (est-ce même possible ?)
